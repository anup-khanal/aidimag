# aiDimag v1.0.0

Initial release of aiDimag - persistent, verified memory for AI coding agents.

## 📦 Installation

### NPM Package (CLI)
```bash
npm install -g aidimag
```

**NPM Package**: https://www.npmjs.com/package/aidimag

### IDE Extensions

Download the extensions below and install manually:

- **VS Code**: Download `aidimag-vscode-1.0.0.vsix` → Install via `code --install-extension aidimag-vscode-1.0.0.vsix`
- **IntelliJ IDEA**: Download `aidimag-intellij-plugin-1.0.0.zip` → Settings → Plugins → Install from Disk

## 📚 Documentation

Full documentation: https://anup-khanal.github.io/aidimag/

## ✨ Features

- Persistent memory for AI coding agents
- Verified, falsifiable claims with evidence
- MCP integration for Claude, Cursor, Copilot
- CLI tool (`dim`)
- VS Code and IntelliJ extensions
- Team sync capabilities
- Knowledge base management
